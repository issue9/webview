======

基于 webview 技术的应用开发框架，目前支持以下平台：

- darwin：支持 macOS 10.13
- windows：采用 webview2，支持 windows 10、windows 11。
- linux：需要安装 WebKit2 2.22 以上版本

安装
----

```shell
go get github.com/issue9/webview
```
